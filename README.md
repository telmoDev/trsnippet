
# TRSnippet README

  

This is an extension that has custom snippets.

  

## Features

  

Language support: PHP, Python, Dart, JavaScript and TypeScript.

  

## PHP

  
- Arrow function

	>prefix: afn

	>fn($x) => $x

- Callback function

	>prefix: cfn
	>function ($x) { return $x; }


- dd by Larabel

	>prefix: ddp
	>dd('Prueba');

  

## Python

 - Function

	> prefix: dfn
	
	> def :
	&nbsp;&nbsp;&nbsp;&nbsp;pass

  

## JS - TS

 - Console.log

	> prefix: cnl
	
	> console.log()
	
- Arrow function

	>prefix: afn

	>(v) => v

- Callback function

	>prefix: cfn
	>function (v) { v }

- Fetch
	>prefix: fet
	
	>fetch('',{
method:''
,body:
}).then(res  => {  })
  
 - Selector

	> prefix: jq
	
	> $('').('');
 - Selector with callback

	> prefix: jqf
	
	> $('').(function  (e)  {});

## Dart

- Arrow function

	>prefix: afn

	>(v) => v

  

- Callback function

	>prefix: cfn
	>(v) { v }

  

**Enjoy!**
